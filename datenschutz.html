2. Übersicht

PlasmidStudio ist eine clientseitige Webanwendung. Alle Daten (Plasmidkarten, Sequenzen, Einstellungen) werden ausschließlich lokal in Ihrem Browser verarbeitet und gespeichert. Es werden keine Nutzerdaten an unsere Server übermittelt, da die Anwendung keinen eigenen Backend-Server betreibt.

3. Hosting

Diese Website wird über GitHub Pages bereitgestellt, einen Dienst der GitHub Inc. (bzw. GitHub B.V. für europäische Nutzer), 88 Colin P. Kelly Jr. Street, San Francisco, CA 94107, USA.

Beim Aufruf dieser Website erhebt GitHub automatisch technische Zugriffsdaten, insbesondere:

  • IP-Adresse
  • Datum und Uhrzeit des Zugriffs
  • Aufgerufene Seite
  • Browser-Typ und Betriebssystem
  • Referrer-URL

Diese Datenverarbeitung erfolgt durch GitHub als Auftragsverarbeiter. Die Rechtsgrundlage ist Art. 6 Abs. 1 lit. f DSGVO (berechtigtes Interesse an der sicheren Bereitstellung der Website). GitHub verarbeitet diese Daten gemäß der GitHub Privacy Statement und hat sich den EU-Standardvertragsklauseln unterworfen.

4. Lokale Datenspeicherung (localStorage)

PlasmidStudio speichert folgende Daten im localStorage Ihres Browsers:

  • Plasmidkarten (Features, Marker, Einstellungen)
  • Farbverlauf (zuletzt verwendete Farben)
  • Theme-Einstellungen
  • Onboarding-Status

Diese Daten verlassen Ihren Browser nicht und werden nicht an Dritte übermittelt. Die Rechtsgrundlage ist Art. 6 Abs. 1 lit. f DSGVO (berechtigtes Interesse an der Funktionsfähigkeit der Anwendung). Sie können diese Daten jederzeit über die Browser-Einstellungen löschen.

5. Externe Verbindungen

5.1 NCBI / AddGene Abruf

Wenn Sie die Funktion „Fetch from NCBI“ oder „Fetch from AddGene“ nutzen, stellt Ihr Browser eine direkte Verbindung zu den Servern des National Center for Biotechnology Information (NCBI, USA) bzw. AddGene (USA) her. Dabei wird Ihre IP-Adresse an diese Dienste übermittelt. Diese Verbindung erfolgt nur auf Ihre ausdrückliche Anforderung.

5.2 PDF-Export

Beim erstmaligen PDF-Export wird die Bibliothek jsPDF von einem CDN (cdnjs.cloudflare.com) geladen. Dabei wird Ihre IP-Adresse an Cloudflare übermittelt. Nach dem ersten Laden wird die Bibliothek vom Service Worker zwischengespeichert.

6. Keine Cookies, kein Tracking

PlasmidStudio verwendet keine Cookies, keine Analyse-Tools und kein Tracking. Es werden keine Werbenetzwerke eingebunden.

7. Schriftarten

Die auf der Startseite verwendete Schriftart (Inter) wird lokal von unserem Server bereitgestellt. Es findet keine Verbindung zu Google Fonts oder anderen externen Schriftarten-Diensten statt.

8. Service Worker

PlasmidStudio verwendet einen Service Worker, um die Anwendung offline verfügbar zu machen. Der Service Worker speichert Anwendungsdateien ausschließlich lokal im Browser-Cache. Es werden keine Daten an externe Server gesendet.

  • Löschung (Art. 17 DSGVO) – Sie können die Löschung Ihrer Daten verlangen.
  • Einschränkung (Art. 18 DSGVO) – Sie können die Einschränkung der Verarbeitung verlangen.

  • Beschwerde – Sie haben das Recht, sich bei einer Datenschutz-Aufsichtsbehörde zu beschweren.

Privacy Policy

Last updated: March 2026

1. Controller

Christopher Acatay
Email: contact@plasmidstudio.app

2. Overview

PlasmidStudio is a client-side web application. All data (plasmid maps, sequences, settings) is processed and stored exclusively in your browser. No user data is transmitted to our servers, as the application does not operate a backend server.

3. Hosting

This website is hosted on GitHub Pages, a service of GitHub Inc. (or GitHub B.V. for European users), 88 Colin P. Kelly Jr. Street, San Francisco, CA 94107, USA.

When you visit this website, GitHub automatically collects technical access data, including:

  • IP address
  • Date and time of access
  • Page visited
  • Browser type and operating system
  • Referrer URL

This data processing is carried out by GitHub as a data processor. The legal basis is Art. 6(1)(f) GDPR (legitimate interest in the secure provision of the website). GitHub processes this data in accordance with the GitHub Privacy Statement and has committed to EU Standard Contractual Clauses.

4. Local Data Storage (localStorage)

PlasmidStudio stores the following data in your browser's localStorage:

  • Plasmid maps (features, markers, settings)
  • Color history (recently used colors)
  • Theme settings
  • Onboarding status

This data never leaves your browser and is not shared with third parties. The legal basis is Art. 6(1)(f) GDPR (legitimate interest in the functionality of the application). You can delete this data at any time via your browser settings.

5. External Connections

5.1 NCBI / AddGene Fetch

When you use the "Fetch from NCBI" or "Fetch from AddGene" feature, your browser connects directly to the servers of the National Center for Biotechnology Information (NCBI, USA) or AddGene (USA). Your IP address is transmitted to these services. This connection is made only at your explicit request.

5.2 PDF Export

On the first PDF export, the jsPDF library is loaded from a CDN (cdnjs.cloudflare.com). Your IP address is transmitted to Cloudflare. After the first load, the library is cached by the service worker.

6. No Cookies, No Tracking

PlasmidStudio does not use cookies, analytics tools, or tracking. No advertising networks are integrated.

7. Fonts

The font used on the landing page (Inter) is served locally from our server. No connection to Google Fonts or other external font services is made.

8. Service Worker

PlasmidStudio uses a service worker to make the application available offline. The service worker stores application files exclusively in the local browser cache. No data is sent to external servers.

9. Your Rights

Under the GDPR, you have the following rights:

  • Access (Art. 15 GDPR) – You may request information about your processed data.
  • Rectification (Art. 16 GDPR) – You may request correction of inaccurate data.
  • Erasure (Art. 17 GDPR) – You may request deletion of your data.
  • Restriction (Art. 18 GDPR) – You may request restriction of processing.
  • Objection (Art. 21 GDPR) – You may object to processing.
  • Complaint – You have the right to lodge a complaint with a data protection supervisory authority.

Since PlasmidStudio does not store personal data on the server side, these rights primarily relate to the access data collected by GitHub Pages.
